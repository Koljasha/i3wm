## i3wm

Установить:
* i3 (i3-gaps - в конфиге)
* i3lock
* i3status
* polybar (в конфиге)
* dmenu
* rofi (в конфиге)
* feh
* compton
* plank (в конфиге)

Установка конфигурации:
* `./install.sh`

Темы для GTK:
* lxappearance

Добавить cron `crontab -e`:
* `* * * * * env DISPLAY=:0.0 feh --randomize --bg-scale ~/Изображения/`
